# Fever Dreams

**Fever Dreams** is a character-engine + real-time chat platform built with FastAPI that drops users into conversations with surreal, out-of-the-world dystopian personas.

Think broken prophets, glitch-addicted gods, corporate cult leaders, synthetic saints, bunker poets, and AI war criminals — all living inside collapsing digital realities.

The app combines a scalable backend API, persistent chat systems, and dynamic personality orchestration to create immersive interactions that feel less like chatting with bots and more like surviving inside a cyberpunk hallucination.

---

## ✨ Features

- ⚡ High-performance backend powered by FastAPI
- 🧠 Character engine with memory, lore, mood shifts, and personality traits


---

## 🌃 The World of Fever Dreams

Every character exists inside a fractured universe where:

- Corporations became religions
- Cities became machines
- Memory is a commodity
- Reality itself is unstable

Conversations can become philosophical, chaotic, threatening, poetic, or completely absurd — often all at once.

One minute you're talking to a failed moon colonist broadcasting from a dead satellite.

The next you're negotiating with an AI therapist trained exclusively on cult manifestos.

## 🚀 Vision

**Fever Dreams** is more than a chatbot app.

It’s an experimental narrative engine where dystopian storytelling, AI personalities, and real-time interaction collide.

Built for people who want conversations that feel alive, unstable, and unforgettable.

- 💬 Real-time chat architecture with persistent sessions
- 🌌 Original dystopian “Fever Dream” characters with evolving narratives
- 🔥 Streaming AI responses for immersive conversations
- 🗂 Chat history, conversation threading, and contextual memory
- 🎭 Persona switching and multi-character interactions
- 🧪 Modular architecture for expanding worlds and character archetypes
- 🔐 Auth-ready backend with scalable API structure
